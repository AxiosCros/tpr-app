# application for use TPR framework 3.0

## Env

* php: 7.1+

## How to use

* install composer library

```bash
composer install
```

* start web server

```bash
composer run
```

> view in brower : [localhost:8088/index.php](http://localhost:8088/index.php)

* use command tool

```shell
chmod 755 ./tpr
./tpr

# generate new command
./tpr make command_name
```
